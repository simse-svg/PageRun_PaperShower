# 2026-06-11 변경 내역

## 1. Home 탭
- 기록 카드의 `PAGERUN` 글자 크기 조정 (28 -> 22)
- 기록 카드 내부 다운로드 버튼 제거
- 기록 카드 클릭 시 열리는 사진 상세 화면에서만 다운로드 버튼 사용

## 2. Mileage 탭
- 달력 화면과 기록 summary 화면 배경을 분홍 파스텔 톤으로 변경
- 달력의 초록 동그라미 크기 규칙 변경
  - 10페이지 이하: 매우 작게
  - 11~99페이지: 중간
  - 100페이지 이상: 크게
- 기록 summary에서 항목 클릭 시 해당 기록의 사진 화면으로 이동

## 3. Record 탭
- 책 이름 드롭다운 선택 시 키보드가 올라오지 않도록 포커스 해제 처리
- 드롭다운으로 책 선택 시 해당 책의 마지막 종료페이지를 시작페이지에 자동 입력
- 메인 사진 선택 다이얼로그
  - 선택 사진 테두리 색상을 흰색에서 빨간색으로 변경
  - `첫 사진 사용` 버튼 제거, `선택 완료` 버튼만 유지
- 하단 4개 버튼(시작/정지/촬영/리셋) 높이를 입력칸과 비슷한 56dp로 조정
- 리셋 버튼 배경색을 다른 버튼과 동일한 빨간색으로 통일
